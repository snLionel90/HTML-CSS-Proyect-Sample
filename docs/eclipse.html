Eclipse

Eclipse IDE

Las aplicaciones para el desarrollo se denominan IDE (Interfaz de entorno de desarrollo)

Eclipse es una plataforma de software compuesto por un conjunto de herramientas de programacion de codigo abierto multiplataforma para desarrollar lo que el proyecto llama "Aplicaciones de Cliente Enriquecido", contrario a las aplicaciones "Cliente-liviano" basadas en navegadores. Esta plataforma, tipicamente ha sido usada para desarrollar entornos de desarrollo integrados (del ingles IDE), como el IDE de Java llamado Java Development Toolkit (JDT) y el compilador (ECJ) que se entrega como parte de Eclipse (y que son usados tambien para desarrollar el mismo Eclipse).


Caracteristicas del IDE

  • Eclipse dispone de un Editor de texto con un analizador sintactico
  • La compilacion es en tiempo real.
  • Tiene pruebas unitarias con JUnit
  • asistentes (wizards) para creacion de proyectos, clases, tests, etc.,
  • Asimismo, a traves de "plugins" libremente disponibles es posible anyadir control de versiones con Subversion.
  • Es multiplataforma
