Intelli J IDEA

IntelliJ IDEA es un entorno de desarrollo integrado para el desarrollo de programas informaticos.

Es desarrollado por JetBrains, y esta disponible en dos ediciones: edicion para la comunidad​ y edicion comercial.

IntelliJ IDEA no esta basada en Eclipse

La version 12.1 incluye soporte para Java 8, diseniador UI para desarrollo de Android,
