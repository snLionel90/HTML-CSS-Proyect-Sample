MY SQL

Es un esntorno de desarrollo o IDE destinado a programacion de Bases de datos

MySQL es un sistema de gestion de bases de datos relacional desarrollado bajo licencia dual: Licencia publica general/Licencia comercial por Oracle Corporation y esta considerada como la base datos de codigo abierto mas popular del mundo, y una de las mas populares en general junto a Oracle y Microsoft SQL Server, sobre todo para entornos de desarrollo web.

Se destaca Por

  • MySQL es muy utilizado en aplicaciones web, como Joomla, Wordpress, Drupal o phpBB,
  • MySQL es una base de datos muy rapida en la lectura cuando utiliza el motor no transaccional
  • MySQL ideal para este tipo de aplicaciones. Sea cual sea el entorno en el que va a utilizar MySQL

CARACTERISTICAS

Entre las caracteristicas disponibles en las ultimas versiones se destacan:

  • Amplio subconjunto del lenguaje SQL.
  • Disponibilidad en gran cantidad de plataformas y sistemas.
  • Posibilidad de seleccion de mecanismos de almacenamiento.
  • Soporte físico, capacidad, distribucion geografica, transacciones, etc.
  • Transacciones y claves ajenas.
  • Conectividad segura.
  • Replicacion.
  • Busqueda e indexacion de campos de texto.
