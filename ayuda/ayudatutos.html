Tutorial Configuracion

Para anadir modulos o plugings desde el IDE hay que ir a Tools Plugings Y de esa ventana seleccionar Available Pluggings y seleccionar los que quieras instalar

Opción 1 para personalizar el IDE con ventanas, simplemente accede a la pestana del menú Windows y selecciona las que crea conveniente, para su espacio de trabajo.

La ventana Files se ubican las clases del proyecto en forma de ficheros La ventana proyectsSe ubican los paquetes con las clases del espacio de trabajo. La Ventana principal se utiliza para escribir rel codigo fuente La ventana Palettees un componente dedicado a mostrar de manera grafica los distintos elementos que pueden ser agregados a la edicion arrastrandolos con el raton La Ventana propierties : Muestra las propiedades de los metodos de una clase. La Ventana Output muestra el proceso de compilacion de las clases y sus metodos del programa

Opcion 2 dentro de el menu Tool-Options, selecciona la pestana Fonts & Colors para configurar la apariencia del IDE a tu gusto

En la pestaña Font& Colors: pesonaliza el IDE a su gusto, asignandole una fuente, tamaño de letra, colores de la fuente y fondo, efectos
